FlowLabLite – Local Results Viewer

Load locally computed simulation results (JSON) for browser visualization

Usage:
1. Run locally: moon run cmd/main --target wasm 2>&1 | sed -n '/===JSON_DATA_START===/,/===JSON_DATA_END===/p' | grep -v '===JSON_DATA' > results.json
2. Or on Windows PowerShell: moon run cmd/main --target wasm 2>&1 | Out-File output.txt; then extract JSON block
3. Load the results.json file below, or paste JSON directly.
